## Usage

**You need to have [Inkscape](https://inkscape.org/) installed.**

### Install python packages
`pip install -r requirements.txt`

### Modify lessons.txt
Format: 
```
Mo1
Mo2
Mo3
Mo4
Mo5
Mo6
Mo7

Di1
Di2
Di3
.
.
.

```
### Run script
`python3 main.py`
